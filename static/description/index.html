Generate serial/lot number based from MO-Manufacturing Odoo Apps

This odoo apps helps to generate serial number and lot number from MO, It is very useful to generate custom serial no for manufacturing order by user.This odoo apps generate automatic serial number and lot number from manufacturing order or from the workorder when manufacturing order finished/done with Manufacturing produce option or done from the workorder. This serial/lot number sequence comes with prefix and sequence defined on the configuration settings.

Features

User can set digits.

You can set number of digits for serial no.

User can set prefix.

You can set number prefix as you wish for serial no.

Helps to make serial/lot number.

You don't have to create serial/lot number manually.

Users don't need to create serial/lot number

It will create new serial/lot number for the product in manufacturing order as per your input.

MO Serial Number Configuration

Under Manufacturing > Configuration > Settings menu you can see serial no configurations.

You have options to set MO Serial Number.

1. Global Option :

If you select "Global" option then in MO, Product lot/serial number is set based on global digit and prefix.

2. Product Wise Option :

If you select "Product Wise" option then in MO, Product lot/serial number is set based on particular product digit and prefix.

Enable "Lots & Serial Numbers"

Make sure that in settings of Inventory "Lots & Serial Numbers" is enable

Tracking as "By Unique Serial Number"

In product select Tracking as "By Unique Serial Number" And create Bill of Materials for the product.

Create Bill of Materials

Create Manufacturing Order

Now you have to click one time produce button and use create batch serial no button in produce wizard.

Produce Wizard

Click on Create Batch Serial No button at once to produce given quantity of product with serial/lot number in sequence.

You can see the finished products with global serial/lot no.

Product Wise MO Serial Number

Under Manufacturing > Configuration > Settings menu you can see serial no, Select Product Wise option.

Set Product Digit and Prefix

Set product digit and prefix under inventory tab.

Create Manufacturing Order

Now you have to click one time produce button and use create batch serial no button in produce wizard.

Produce Wizard

Click on Create Batch Serial No button at once to produce given quantity of product with serial/lot number in sequence.

You can see the finished products with product wise serial/lot no.

Configurations

User have to select "Serial Number Apply Method" and enter digits and prefix for serial no.

Tracking as "By Unique Serial Number" in Work Order

Select product tracking as "By Unique Serial Number" under Traceability.

Bill of Materials

Create and select "Routing" on bill of materials.

Manufacturing Order

User can see created work order in tree view for manufacturing order.

Auto Generated Serial Number for each Quantity

User can see auto generated serial number for produced quantity of both work order.

User can see auto created serial number for manufacturing product under "Finished Product" tab on MO.

Tracking as "By Lots"

User have to select "By Lots" in product tracking under traceability.

Manufacturing Order

User can see create work order in tree view for manufacturing order.

Auto Generated Lots

User can see auto generated lots number for produced quantity both work order.

User can see auto generated lots number for finished product.
